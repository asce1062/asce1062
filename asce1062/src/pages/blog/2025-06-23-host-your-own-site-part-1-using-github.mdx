# PT.1 Webbed Site Repository
<br/>
- Create a [new](https://github.com/new) GitHub repository
- Repository name: _**`githubusername`**_
- <sup>*</sup>_**`Public`**_
- Check _**` Initialize this repository with: Add a README file`**_
- Create repository
<br/>
<ImageView image = "/src/assets/blog/github/github-repo-setup.png" altText="GitHub Repository Setup"></ImageView>
<br/>
- Edit _**`README.md`**_ file
- For github markdown syntax, visit this [getting started doc](https://docs.github.com/en/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax)
<br/>
<ImageView image = "/src/assets/blog/github/github-edit-readme.png" altText="Edit README.md"></ImageView>
<br/>
- For formatting that isn't supported by  _**`markdown's syntax`**_ or for more complex layouts, you can use _**`HTML tags`**_ directly in _**`markdown`**_
- Check out my [README.md](https://github.com/asce1062/asce1062/blob/main/README.md) for inspiration if needed (switch to _**`code view`**_ for markdown html syntax)
<br/>
<ImageView image = "/src/assets/blog/github/github-html-in-markdown.png" altText="HTML in Markdown"></ImageView>
<br/>
- Save your changes by _**`Committing changes...`**_
<br/>
<ImageView image = "/src/assets/blog/github/github-commit-readme.png" altText="Commit changes..."></ImageView>
<br/>
- Navigate to repository's _**`Settings > Pages`**_
- Under _**`Build and deployment Source`**_ select _**`Deploy from a branch`**_
- Under _**`Branch`**_ select _**`main`**_
- Under _**`select folder`**_ _**`/(root)`**_
- _**`Save`**_ these changes
<br/>
<ImageView image = "/src/assets/blog/github/github-pages-setup.png" altText="GitHub Pages Setup"></ImageView>
<br/>
- A build should be automatically triggered
- Progress can be tracked under the repository's _**`Actions`**_ tab
<br/>
<ImageView image = "/src/assets/blog/github/github-actions-tab.png" altText="GitHub Actions tab"></ImageView>
<br/>
- Once deployment is done, navigate back to repository's _**`Settings > Pages`**_
- You should see _**`Your site is live at`**_ _**`https://username.github.io/reponame/`**_
- Since we created our _**`repositoryname`**_ like: _**`githubusername`**_, the expected _**`site url`**_ is _**`https://githubname.github.io/githubname/`**_
- In part 2 we'll cover <a href="/blog/2025-06-24-host-your-own-site-part-2-adding-a-theme-with-jekyll" aria-label="Visit the blog page">adding a theme with Jekyll</a> to our webbed site
</span>
