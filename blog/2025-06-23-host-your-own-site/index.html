Host Your Own Site

2025-06-23 | Part 1 - Using GitHub tags: githubhowtohosting

PT.1 Webbed Site Repository

  • Create a new GitHub repository
  • Repository name: githubusername
  • *Public
  • Check Initialize this repository with: Add a README file
  • Create repository
  • Edit README.md file https://github.com/githubusername/githubusername/edit/main/README.md
  • For github markdown syntax, visit this getting started doc
  • For formatting that isn’t supported by markdown's syntax or for more complex layouts, you can use HTML tags directly in your markdown.
  • Check out my README.md for inspiration if needed (code view for markdown html syntax)
  • Save your changes by Committing changes...
  • Navigate to repository’s Settings > Pages https://github.com/githubusername/githubusername/settings/pages
  • Under Build and deployment Source select Deploy from a branch
  • Under Branch select main
  • Under select folder /(root)
  • Save these changes
  • An automatic build should be triggered
  • Progress can be tracked under the repository’s Actions tab https://github.com/githubusername/githubusername/actions
  • Once deployment is done, navigate back to repository’s Settings > Pages https://github.com/githubusername/githubusername/settings/pages
  • You should see Your site is live at https://githubusername.github.io/repositoryname/
  • Since we created our repositoryname like: githubusername, the site url expected is https://githubusername.github.io/githubusername/
